He creat un mini arcade en Python amb un menu principal que permet triar entre dos jocs: Pedra, Paper o Tisora i Endevinar el Numero. He programat cada joc dins d’una funcio i he organitzat el codi en diferents fitxers per tenir-lo mes ordenat. Amb aquest projecte he practicat l’us de bucles, condicions, variables i el modul "random" per fer el programa interactiu.
